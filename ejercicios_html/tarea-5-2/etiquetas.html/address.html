La Etiqueta ADDRESS

Volver al inicio

¿Qué es la etiqueta ADDRESS?

La etiqueta <address> proporciona información de contacto del autor o propietario de un documento o artículo. No se utiliza para direcciones postales genéricas.

Descripcion

La etiqueta <address> es un elemento semántico que identifica información de contacto relacionada con el autor o responsable del contenido más cercano (<article> o <body>).

Información apropiada para <address>:

  • Dirección de correo electrónico del autor
  • Enlace al perfil del autor
  • Dirección física de una organización responsable del contenido
  • Número de teléfono de contacto del autor
  • Redes sociales del autor o empresa

Importante: No debe usarse para cualquier dirección postal en la página. Solo para información de contacto del autor o propietario del contenido. Por ejemplo, no usar <address> para la dirección de envío en un formulario de compra.

Atributos principales

La etiqueta <address> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Por defecto, los navegadores suelen mostrar el contenido de <address> en cursiva.

Ejemplo práctico

<footer>
  <address>
    Contacto: <a href="mailto:info@ejemplo.com">info@ejemplo.com</a><br>
    Teléfono: +34 123 456 789<br>
    Calle Ejemplo 123, Madrid, España
  </address>
</footer>

Notas importantes

  • Puede haber múltiples <address> en un documento (una por autor/propietario)
  • Comúnmente se coloca dentro de <footer>
  • No puede contener elementos de encabezado (h1-h6), <header>, <footer>, <section>, <article> u otro <address>
  • Mejora el SEO y la accesibilidad al identificar claramente información de contacto
  • Solo para contacto del autor/propietario, no para direcciones arbitrarias en el contenido
