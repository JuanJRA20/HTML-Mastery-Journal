La Etiqueta SPAN

Volver al inicio

¿Qué es la etiqueta SPAN?

La etiqueta <span> es un contenedor en línea genérico sin significado semántico. Se utiliza para aplicar estilos o manipular con JavaScript porciones específicas de texto.

Descripcion

La etiqueta <span> es un elemento en línea neutral que no tiene significado semántico por sí mismo. Es el equivalente en línea de <div> (que es de bloque). Se usa principalmente como gancho para aplicar estilos CSS o JavaScript a partes específicas del texto.

Usos comunes de <span>:

  • Aplicar color o estilos a palabras específicas
  • Cambiar la fuente de una parte del texto
  • Añadir iconos o símbolos dentro de texto
  • Marcar texto para manipulación con JavaScript
  • Aplicar clases CSS a fragmentos de texto

Regla importante: Solo usa <span> cuando no exista una etiqueta semántica más apropiada. Por ejemplo, usa <strong> para importancia, <em> para énfasis, <code> para código, etc. <span> es el último recurso.

Atributos principales

La etiqueta <span> no tiene atributos específicos propios, pero acepta todos los atributos globales de HTML5, siendo los más comunes:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos (el más usado)
  • style - Estilos CSS inline

De hecho, <span> raramente se usa sin atributos, ya que su propósito principal es servir como contenedor para aplicar estilos o scripts.

Ejemplo práctico

<p>
  El precio es <span class="precio">29,99€</span> 
  con descuento.
</p>

<p>
  Mi color favorito es el 
  <span style="color: blue;">azul</span>.
</p>

<p>
  Haz clic <span id="contador">0</span> veces.
</p>

Notas importantes

  • Es un elemento en línea (no genera salto de línea)
  • No tiene significado semántico, es puramente para presentación
  • Preferir etiquetas semánticas cuando sea posible (<strong>, <em>, <code>, etc.)
  • Muy utilizado en combinación con clases CSS
  • Es el contenedor más pequeño y específico en HTML
  • Por defecto, no aplica ningún estilo visual
