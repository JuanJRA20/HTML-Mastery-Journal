La Etiqueta LABEL

Volver al inicio

¿Qué es la etiqueta LABEL?

La etiqueta <label> define una etiqueta de texto para un control de formulario, mejorando la accesibilidad y usabilidad al asociar texto descriptivo con campos de entrada.

Descripcion

La etiqueta <label> es un elemento en línea fundamental para la accesibilidad de formularios. Asocia texto descriptivo con controles como <input>, <textarea>, <select>, etc., mejorando enormemente la experiencia del usuario.

Beneficios de usar <label>:

  • Mejora la accesibilidad para lectores de pantalla
  • Aumenta el área clickeable (hacer clic en el label activa el control)
  • Proporciona contexto claro sobre qué datos se esperan
  • Es esencial para cumplir estándares de accesibilidad web (WCAG)
  • Facilita el uso en dispositivos móviles (área de toque más grande)

Regla de oro: Cada campo de entrada debería tener un <label> asociado. No usar placeholders como sustituto de labels, ya que desaparecen al escribir.

Atributos principales

A continuación se explican en detalle los atributos más importantes:

for

Asocia el label con un control específico mediante el id del control. Es la forma más común y robusta de vincular label y campo. El valor debe coincidir exactamente con el id del elemento de formulario.

Ejemplo: <label for="nombre">Nombre:</label> <input type="text" id="nombre">

Ejemplo práctico

<!-- Método recomendado: usando for + id -->
<label for="email">Correo electrónico:</label>
<input type="email" id="email" name="email">

<!-- Método alternativo: anidando el control -->
<label>
  Nombre:
  <input type="text" name="nombre">
</label>

<!-- Labels con checkboxes -->
<input type="checkbox" id="terminos" name="terminos">
<label for="terminos">Acepto los términos y condiciones</label>

<!-- Labels con radio buttons -->
<input type="radio" id="hombre" name="genero" value="hombre">
<label for="hombre">Hombre</label>

<input type="radio" id="mujer" name="genero" value="mujer">
<label for="mujer">Mujer</label>

<!-- Label con textarea -->
<label for="mensaje">Mensaje:</label>
<textarea id="mensaje" name="mensaje" rows="5"></textarea>

<!-- Label con select -->
<label for="pais">País:</label>
<select id="pais" name="pais">
  <option value="es">España</option>
  <option value="mx">México</option>
</select>

<!-- Formulario completo con labels -->
<form>
  <div>
    <label for="usuario">Usuario:</label>
    <input type="text" id="usuario" name="usuario" required>
  </div>
  
  <div>
    <label for="password">Contraseña:</label>
    <input type="password" id="password" name="password" required>
  </div>
  
  <div>
    <input type="checkbox" id="recordar" name="recordar">
    <label for="recordar">Recordarme</label>
  </div>
  
  <button type="submit">Iniciar Sesión</button>
</form>

Notas importantes

  • Requiere etiqueta de cierre </label>
  • El atributo for debe coincidir exactamente con el id del control
  • Hacer clic en el label activa/enfoca el control asociado
  • Para checkboxes y radios, hacer clic en el label marca/desmarca la casilla
  • Hay dos formas de asociar: con for/id (preferida) o anidando el control dentro del label
  • No usar placeholder como sustituto de label (problemas de accesibilidad)
  • Es obligatorio para accesibilidad web según estándares WCAG
  • Mejora enormemente la usabilidad en dispositivos táctiles
