La Etiqueta WBR

Volver al inicio

¿Qué es la etiqueta WBR?

La etiqueta <wbr> (Word Break Opportunity) indica un punto donde el navegador puede opcionalmente romper una línea si es necesario, pero no fuerza el salto.

Descripcion

La etiqueta <wbr> es un elemento vacío introducido en HTML5 que sugiere un punto de ruptura opcional en palabras o URLs muy largas. A diferencia de <br>, que siempre rompe la línea, <wbr> solo lo hace si el navegador necesita ajustar el texto al ancho disponible.

Usos apropiados de <wbr>:

  • URLs muy largas que podrían desbordar el contenedor
  • Palabras compuestas extremadamente largas
  • Rutas de archivos largas
  • Direcciones de correo electrónico muy largas
  • Código o identificadores largos sin espacios naturales

Funcionamiento: Si la palabra cabe en la línea, <wbr> no hace nada. Si no cabe, el navegador puede romper en ese punto sin añadir guion.

Atributos principales

La etiqueta <wbr> no tiene atributos específicos. Es una etiqueta vacía que raramente necesita atributos, aunque técnicamente acepta atributos globales:

  • id - Identificador único (raramente usado)
  • class - Clase CSS (raramente usado)

En la práctica, <wbr> casi nunca lleva atributos.

Ejemplo práctico

<p>
  Visita nuestra web en 
  http://<wbr>www.<wbr>ejemplo<wbr>muylargoynecesita<wbr>romper.<wbr>com
</p>

<p>
  Archivo: C:\<wbr>Usuarios\<wbr>Documentos\<wbr>MiCarpeta\<wbr>archivo.txt
</p>

Notas importantes

  • Es una etiqueta vacía (no tiene </wbr> de cierre)
  • Introducida en HTML5, no funciona en navegadores muy antiguos
  • No añade guion al romper la línea (a diferencia del CSS hyphen)
  • Solo rompe si es necesario por espacio, no siempre
  • Muy útil en diseño responsive para evitar desbordes horizontales
  • Es una de las etiquetas más especializadas y menos utilizadas de HTML5
