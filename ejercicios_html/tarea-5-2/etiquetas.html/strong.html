La Etiqueta STRONG

Volver al inicio

¿Qué es la etiqueta STRONG?

La etiqueta <strong> indica que el texto tiene una importancia fuerte o especial. Por defecto, los navegadores lo muestran en negrita.

Descripcion

La etiqueta <strong> es un elemento semántico en línea que marca texto con alta importancia, seriedad o urgencia. A diferencia de <b> (que solo aplica negrita visual), <strong> tiene significado semántico.

Usos apropiados de <strong>:

  • Advertencias importantes
  • Términos clave que deben destacarse
  • Información crítica que el usuario debe notar
  • Palabras o frases de alta relevancia en el contexto

Diferencia clave: Usa <strong> cuando el texto es importante para el significado. Usa <b> cuando solo quieras negrita por estilo visual sin importancia semántica. Los lectores de pantalla enfatizan el contenido de <strong>.

Atributos principales

La etiqueta <strong> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Aunque raramente se necesita aplicar atributos a <strong>, pueden usarse para estilos CSS específicos.

Ejemplo práctico

<p>
  <strong>Advertencia:</strong> No uses este producto 
  cerca del agua.
</p>

<p>
  El concepto más importante es la 
  <strong>semántica</strong> en HTML5.
</p>

Notas importantes

  • Es un elemento en línea (no genera salto de línea)
  • Tiene significado semántico, no es solo para estilo visual
  • Los lectores de pantalla enfatizan el texto dentro de <strong>
  • Puede anidarse con <em> para importancia + énfasis
  • No usar en exceso; si todo es importante, nada lo es
  • Preferir <strong> sobre <b> en HTML5 moderno
