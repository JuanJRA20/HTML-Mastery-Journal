La Etiqueta HEAD

Volver al inicio

¿Qué es la etiqueta HEAD?

La etiqueta <head> es el contenedor de metadatos del documento HTML. Contiene información sobre la página que no se muestra directamente al usuario, pero que es esencial para el navegador, buscadores y otros servicios.

Descripcion

La etiqueta <head> se coloca siempre después de la etiqueta de apertura <html> y antes de <body>. Todo lo que está dentro del <head> es información "invisible" para el visitante de la página, pero crucial para su correcto funcionamiento.

Dentro del <head> se incluyen elementos como:

  • <title> - El título que aparece en la pestaña del navegador
  • <meta> - Metadatos diversos (charset, viewport, descripción, etc.)
  • <link> - Enlaces a hojas de estilo CSS y otros recursos externos
  • <script> - Referencias a archivos JavaScript (aunque también pueden ir en el body)
  • <style> - Estilos CSS internos

Atributos principales

La etiqueta <head> acepta los siguientes atributos globales:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

En la práctica, estos atributos casi nunca se utilizan en la etiqueta <head>.

Ejemplo práctico

<head>
  <meta charset="UTF-8">
  <title>Mi Página Web</title>
  <link rel="stylesheet" href="estilos.css">
</head>

Notas importantes

  • Debe haber solo una etiqueta <head> por documento
  • Es obligatoria en todo documento HTML válido
  • Debe contener al menos un elemento <title>
  • El contenido del <head> no es visible en la página, pero afecta su comportamiento y apariencia
  • Es donde se definen metadatos críticos para SEO y redes sociales
