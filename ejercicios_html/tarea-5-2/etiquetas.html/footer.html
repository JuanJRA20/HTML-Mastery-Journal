La Etiqueta FOOTER

Volver al inicio

¿Qué es la etiqueta FOOTER?

La etiqueta <footer> representa el pie de página de un documento o de una sección. Típicamente contiene información sobre el autor, copyright, enlaces relacionados o información de contacto.

Descripcion

La etiqueta <footer> es un elemento semántico introducido en HTML5 que se utiliza para agrupar contenido de cierre o información complementaria al final de un documento o sección.

Contenido típico de un <footer>:

  • Información de copyright y derechos de autor
  • Enlaces a políticas de privacidad y términos de uso
  • Información de contacto
  • Enlaces a redes sociales
  • Mapa del sitio
  • Información sobre el autor de un artículo
  • Fecha de publicación o última actualización

Al igual que <header>, puede usarse múltiples veces: un footer general de la página y footers dentro de elementos como <article> o <section>.

Atributos principales

La etiqueta <footer> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos se utilizan frecuentemente para aplicar estilos CSS específicos al pie de página.

Ejemplo práctico

<footer>
  <p>© 2026 Mi Sitio Web. Todos los derechos reservados.</p>
  <nav>
    <a href="privacidad.html">Política de Privacidad</a>
    <a href="contacto.html">Contacto</a>
  </nav>
</footer>

Notas importantes

  • Puede haber múltiples <footer> en un mismo documento
  • No puede estar dentro de otro <footer> ni dentro de <header>
  • No necesariamente debe estar al final del documento (puede ser footer de una sección)
  • Mejora la accesibilidad identificando claramente información de cierre
  • Es diferente de estar simplemente "al final": debe contener información típica de pie de página
