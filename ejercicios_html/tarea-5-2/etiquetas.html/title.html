¿Qué es la etiqueta TITLE?

La etiqueta <title> define el título del documento HTML que aparece en la pestaña del navegador y en los resultados de búsqueda.

Descripcion

La etiqueta <title> es obligatoria y debe colocarse dentro del <head>. El texto que contiene no aparece en el cuerpo de la página, pero es visible en:

  • La pestaña o ventana del navegador
  • Los resultados de motores de búsqueda (como Google)
  • Los favoritos o marcadores del navegador
  • El historial de navegación

Es uno de los elementos más importantes para SEO (optimización en motores de búsqueda).

Atributos principales

La etiqueta <title> no tiene atributos específicos. Solo contiene texto plano, sin etiquetas HTML dentro.

Ejemplo práctico

<head>
  <title>Mi Página Web - Inicio</title>
</head>

Notas importantes

  • Es obligatoria en todo documento HTML válido
  • Solo puede haber una etiqueta <title> por documento
  • Debe contener solo texto, no puede incluir otras etiquetas HTML
  • Se recomienda mantenerlo entre 50-60 caracteres para SEO
  • Debe ser descriptivo y único para cada página del sitio
