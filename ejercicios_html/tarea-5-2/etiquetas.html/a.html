La Etiqueta A

Volver al inicio

¿Qué es la etiqueta A?

La etiqueta <a> (anchor/ancla) crea hipervínculos que permiten navegar a otras páginas web, recursos, secciones del mismo documento o iniciar descargas.

Descripcion

La etiqueta <a> es uno de los elementos más fundamentales de la web, ya que crea los enlaces que conectan documentos entre sí. Es un elemento en línea que puede contener texto, imágenes u otros elementos en línea.

Tipos de enlaces con <a>:

  • Enlaces a otras páginas web (URLs externas o internas)
  • Enlaces a secciones dentro de la misma página (anclas con #id)
  • Enlaces para enviar correos electrónicos (mailto:)
  • Enlaces para realizar llamadas telefónicas (tel:)
  • Enlaces para descargar archivos

Elemento fundamental: Sin la etiqueta <a>, la World Wide Web tal como la conocemos no existiría. Los hipervínculos son la base de la navegación web.

Atributos principales

href

Especifica la URL de destino del enlace. Es el atributo más importante y casi siempre está presente. Puede ser una URL completa, una ruta relativa, un ancla (#id), o protocolos especiales como mailto: o tel:.

Ejemplo: <a href="https://ejemplo.com">Enlace</a>

target

Define dónde se abrirá el enlace. Los valores más comunes son _blank (nueva pestaña/ventana), _self (misma ventana, por defecto), _parent y _top.

Ejemplo: <a href="pagina.html" target="_blank">Abrir en nueva pestaña</a>

rel

Define la relación entre el documento actual y el enlazado. Valores importantes: nofollow (no seguir para SEO), noopener (seguridad con target="_blank"), noreferrer (no enviar información de origen), external (enlace externo).

Ejemplo: <a href="externo.com" target="_blank" rel="noopener noreferrer">Enlace seguro</a>

download

Indica que al hacer clic se descargará el recurso en lugar de navegar a él. Opcionalmente puede especificar el nombre de archivo sugerido para la descarga.

Ejemplo: <a href="documento.pdf" download="mi-documento.pdf">Descargar PDF</a>

title

Proporciona información adicional sobre el enlace que aparece como tooltip al pasar el mouse. Útil para accesibilidad y contexto adicional.

Ejemplo: <a href="contacto.html" title="Ir a la página de contacto">Contacto</a>

hreflang

Indica el idioma del documento enlazado. Útil para sitios multiidioma y mejora el SEO internacional.

Ejemplo: <a href="about-en.html" hreflang="en">About us</a>

Ejemplo práctico

<!-- Enlace simple -->
<a href="contacto.html">Contacto</a>

<!-- Enlace externo en nueva pestaña -->
<a href="https://ejemplo.com" target="_blank" rel="noopener noreferrer">
  Visitar sitio externo
</a>

<!-- Enlace a sección de la página -->
<a href="#seccion-importante">Ir a sección</a>

<!-- Enlace de correo -->
<a href="mailto:info@ejemplo.com">Enviar correo</a>

<!-- Enlace de descarga -->
<a href="archivo.pdf" download>Descargar PDF</a>

Notas importantes

  • Es un elemento en línea pero puede contener elementos de bloque en HTML5
  • Sin href, técnicamente es un "marcador de posición" para enlace futuro
  • Siempre usa rel="noopener noreferrer" con target="_blank" por seguridad
  • Los enlaces deben tener texto descriptivo, no "haz clic aquí" (accesibilidad)
  • Pueden contener imágenes: <a href="..."><img src="..."></a>
  • Los navegadores aplican estilos por defecto (azul, subrayado para no visitados)
