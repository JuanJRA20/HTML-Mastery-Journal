¿Qué es la etiqueta HEADER?

La etiqueta <header> representa el encabezado de un documento o de una sección. Contiene típicamente elementos introductorios o de navegación.

Descripcion

La etiqueta <header> es un elemento semántico introducido en HTML5. Se utiliza para agrupar contenido introductorio, como:

  • El logo del sitio web
  • El título principal de la página o sección
  • Menús de navegación principales
  • Información del autor o fecha de publicación
  • Breadcrumbs (migas de pan)

Puede usarse múltiples veces en un documento: una para el encabezado principal de la página y otras dentro de elementos como <article> o <section>.

Atributos principales

La etiqueta <header> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos se usan frecuentemente para aplicar estilos CSS o manipular el elemento con JavaScript.

Ejemplo práctico

<header>
  <img src="logo.png" alt="Logo del sitio">
  <h1>Mi Sitio Web</h1>
  <nav>
    <a href="inicio.html">Inicio</a>
    <a href="contacto.html">Contacto</a>
  </nav>
</header>

Notas importantes

  • No confundir con <head> (que contiene metadatos)
  • Puede haber múltiples <header> en un mismo documento
  • No puede estar dentro de <footer>, <address> u otro <header>
  • Mejora la accesibilidad y el SEO al estructurar semánticamente el contenido
  • Es un elemento de bloque que agrupa contenido, no solo para estética
