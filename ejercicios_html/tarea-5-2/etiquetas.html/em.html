La Etiqueta EM

Volver al inicio

¿Qué es la etiqueta EM?

La etiqueta <em> indica énfasis semántico en el texto. Por defecto, los navegadores lo muestran en cursiva (itálica).

Descripcion

La etiqueta <em> es un elemento semántico en línea que marca texto con énfasis. A diferencia de <i> (que solo aplica cursiva visual), <em> tiene significado semántico que indica que el texto debe pronunciarse o leerse con énfasis especial.

Usos apropiados de <em>:

  • Palabras que se pronunciarían con énfasis al hablar
  • Términos técnicos o extranjeros que necesitan destacarse
  • Cambios sutiles en el significado según qué palabra se enfatice
  • Matices o ironía en el texto

Ejemplo de cambio de significado:
"Yo no dije que robaras el dinero" (otra persona lo dijo)
"Yo no dije que robaras el dinero" (no fue robo, quizás fue prestado)

Atributos principales

La etiqueta <em> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Generalmente no se necesitan atributos en <em>, aunque pueden usarse para estilos CSS personalizados.

Ejemplo práctico

<p>
  No puedo <em>creer</em> que hayas hecho eso.
</p>

<p>
  El término <em>semantic web</em> se refiere a 
  la web con significado.
</p>

<p>
  <em>Yo</em> no rompí el jarrón.
</p>

Notas importantes

  • Es un elemento en línea (no genera salto de línea)
  • Tiene significado semántico de énfasis, no es solo estilo visual
  • Los lectores de pantalla cambian la entonación al leer contenido de <em>
  • Puede anidarse: <em><em>texto</em></em> indica énfasis mayor
  • Diferente de <strong>: <em> es énfasis de pronunciación, <strong> es importancia
  • Preferir <em> sobre <i> en HTML5 cuando el contexto es énfasis
