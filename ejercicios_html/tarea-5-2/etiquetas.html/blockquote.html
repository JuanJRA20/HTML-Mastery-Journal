La Etiqueta BLOCKQUOTE

Volver al inicio

¿Qué es la etiqueta BLOCKQUOTE?

La etiqueta <blockquote> representa una cita extensa proveniente de otra fuente. Se utiliza para citar bloques de texto de otros autores o documentos.

Descripcion

La etiqueta <blockquote> es un elemento de bloque semántico que indica que el contenido es una cita textual de una fuente externa. Los navegadores típicamente aplican sangría (indentación) al contenido para distinguirlo visualmente del resto del texto.

Características de <blockquote>:

  • Para citas largas que ocupan uno o más párrafos
  • Puede contener múltiples elementos dentro (párrafos, listas, etc.)
  • Se muestra como elemento de bloque (con márgenes y sangría)
  • Debe usarse solo para contenido realmente citado, no para sangría decorativa

Nota: Para citas cortas dentro de un párrafo, usa <q> en lugar de <blockquote>.

Atributos principales

cite

Especifica la URL de la fuente de donde proviene la cita. Este atributo no es visible para el usuario, pero es útil para referencias y accesibilidad.

Ejemplo: <blockquote cite="https://ejemplo.com/articulo">

Ejemplo práctico

<blockquote cite="https://www.w3.org/standards/">
  <p>
    La Web está diseñada para funcionar para todas las 
    personas, sea cual sea su hardware, software, idioma, 
    ubicación o capacidad.
  </p>
  <footer>— W3C, <cite>Web Standards</cite></footer>
</blockquote>

Notas importantes

  • Es un elemento de bloque (genera salto de línea)
  • Los navegadores aplican sangría por defecto
  • El atributo cite debe contener una URL válida
  • Puede contener un <footer> con información de la fuente
  • Usar junto con <cite> para referenciar el título de la obra citada
  • No usar solo para crear sangría visual (usa CSS en su lugar)
