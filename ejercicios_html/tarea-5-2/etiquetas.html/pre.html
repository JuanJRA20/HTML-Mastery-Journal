La Etiqueta PRE

Volver al inicio

¿Qué es la etiqueta PRE?

La etiqueta <pre> representa texto preformateado que preserva exactamente los espacios, tabulaciones y saltos de línea tal como aparecen en el código HTML.

Descripcion

La etiqueta <pre> es un elemento de bloque que muestra el contenido exactamente como está escrito, respetando todos los espacios en blanco y saltos de línea. Los navegadores muestran el contenido con una fuente monoespaciada por defecto.

Usos apropiados de <pre>:

  • Bloques de código de programación (generalmente con <code> dentro)
  • Poesía o texto que requiere formato específico de líneas
  • Arte ASCII
  • Salida de terminal o consola
  • Diagramas de texto
  • Cualquier contenido donde el espaciado y formato exacto sean importantes

Comportamiento clave: A diferencia de otros elementos HTML que colapsan múltiples espacios en uno, <pre> mantiene todos los espacios y saltos de línea exactamente como están escritos.

Atributos principales

La etiqueta <pre> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Comúnmente se usa class para aplicar estilos o temas de resaltado de sintaxis.

Ejemplo práctico

<pre>
function saludar() {
    console.log("Hola Mundo");
    return true;
}
</pre>

<pre><code>
#!/bin/bash
echo "Iniciando script..."
cd /home/usuario
ls -la
</code></pre>

Notas importantes

  • Es un elemento de bloque (genera salto de línea)
  • Preserva todos los espacios, tabulaciones y saltos de línea
  • Se muestra con fuente monoespaciada por defecto
  • Frecuentemente se combina con <code> para bloques de código: <pre><code>...</code></pre>
  • El texto dentro puede desbordar horizontalmente si las líneas son muy largas
  • Útil para mantener el formato exacto sin necesidad de CSS complejo
