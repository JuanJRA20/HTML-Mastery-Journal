La Etiqueta CITE

Volver al inicio

¿Qué es la etiqueta CITE?

La etiqueta <cite> marca el título de una obra creativa (libro, película, canción, artículo, etc.). Por defecto, los navegadores lo muestran en cursiva.

Descripcion

La etiqueta <cite> es un elemento semántico en línea que se utiliza para referenciar el título de una obra. Es importante notar que <cite> es para el título de la obra, NO para el nombre del autor.

Usos apropiados de <cite>:

  • Títulos de libros
  • Nombres de películas o series
  • Títulos de canciones o álbumes
  • Nombres de artículos o papers académicos
  • Títulos de obras de arte
  • Nombres de sitios web o blogs
  • Títulos de obras de teatro

Importante: NO uses <cite> para nombres de personas. Es solo para títulos de obras creativas.

Atributos principales

La etiqueta <cite> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Generalmente no se necesitan atributos en <cite>, aunque pueden usarse para estilos CSS personalizados.

Ejemplo práctico

<p>
  Mi libro favorito es 
  <cite>Cien años de soledad</cite> 
  de Gabriel García Márquez.
</p>

<blockquote>
  <p>Ser o no ser, esa es la cuestión.</p>
  <footer>
    — William Shakespeare, <cite>Hamlet</cite>
  </footer>
</blockquote>

Notas importantes

  • Es un elemento en línea (no genera salto de línea)
  • Por defecto se muestra en cursiva
  • Solo para títulos de obras, NO para nombres de autores
  • Comúnmente se usa dentro de <blockquote> para citar la fuente
  • Mejora la semántica y accesibilidad del documento
  • Puede combinarse con enlaces <a> si la obra está disponible online
