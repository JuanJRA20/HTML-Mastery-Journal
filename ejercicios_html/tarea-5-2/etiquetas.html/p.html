¿Qué es la etiqueta P?

La etiqueta <p> define un párrafo de texto. Es uno de los elementos más básicos y utilizados en HTML para estructurar contenido textual.

Descripcion

La etiqueta <p> es un elemento de bloque que representa un párrafo. Los navegadores automáticamente añaden espacio antes y después de cada párrafo para separarlos visualmente.

Características de <p>:

  • Es un elemento de bloque (ocupa todo el ancho disponible)
  • Añade márgenes automáticos arriba y abajo
  • Puede contener texto y elementos en línea (como <strong>, <em>, <a>, etc.)
  • No puede contener otros elementos de bloque (como otro <p>, <div>, o <h1>)

Nota importante: Los saltos de línea y espacios múltiples en el código HTML se reducen a un solo espacio en la visualización. Para forzar saltos de línea, usa <br>.

Atributos principales

La etiqueta <p> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos son muy utilizados para aplicar estilos específicos a párrafos individuales o grupos de párrafos.

Ejemplo práctico

<p>Este es un párrafo simple de texto.</p>

<p>Este es otro párrafo que contiene 
<strong>texto en negrita</strong> y 
<em>texto en cursiva</em>.</p>

<p class="destacado">Este párrafo tiene una clase CSS.</p>

Notas importantes

  • Es una de las etiquetas más fundamentales y utilizadas en HTML
  • Los espacios en blanco múltiples y saltos de línea se colapsan en uno solo
  • No usar <p> vacíos para crear espacios verticales (usa CSS para eso)
  • Los navegadores cierran automáticamente un <p> si encuentran otro elemento de bloque dentro
  • Para mantener espacios y saltos de línea, usa <pre> en lugar de <p>
