Las Etiquetas H1 a H6

¿Qué son las etiquetas H1 a H6?

Las etiquetas <h1> hasta <h6> definen encabezados o títulos en HTML, organizados jerárquicamente del más importante (h1) al menos importante (h6).

Descripcion

Los elementos de encabezado establecen una jerarquía de información en el documento. Son fundamentales para la estructura y legibilidad del contenido.

Jerarquía de encabezados:

  • <h1> - Título principal, el más importante (generalmente uno por página)
  • <h2> - Secciones principales del contenido
  • <h3> - Subsecciones dentro de h2
  • <h4> - Subsecciones dentro de h3
  • <h5> - Subsecciones dentro de h4
  • <h6> - El nivel más bajo de encabezado

Regla de oro: La jerarquía debe ser lógica y consecutiva. No saltes de h1 a h4 sin pasar por h2 y h3. Los encabezados crean la "tabla de contenidos" implícita de tu página.

Atributos principales

Las etiquetas de encabezado no tienen atributos específicos propios. Aceptan los atributos globales de HTML5:

  • id - Identificador único, útil para crear enlaces de anclaje
  • class - Clase CSS para aplicar estilos personalizados

El atributo id es especialmente útil en encabezados para crear tablas de contenidos navegables.

Ejemplo práctico

<h1>Guía de HTML5</h1>
<h2>Introducción</h2>
<p>Contenido introductorio...</p>

<h2>Elementos Básicos</h2>
<h3>Etiquetas de Texto</h3>
<p>Descripción...</p>

<h3>Etiquetas de Enlace</h3>
<p>Descripción...</p>

Notas importantes

  • Solo debe haber un <h1> por página (buena práctica para SEO)
  • No usar encabezados solo para cambiar el tamaño de texto (usa CSS para eso)
  • No saltar niveles de jerarquía (ej: de h2 a h5 directamente)
  • Los motores de búsqueda dan mucha importancia a los encabezados, especialmente h1
  • Los lectores de pantalla usan los encabezados para navegar por el contenido
  • Los navegadores aplican estilos por defecto (h1 más grande, h6 más pequeño)
