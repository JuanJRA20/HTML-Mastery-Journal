La Etiqueta CODE

Volver al inicio

¿Qué es la etiqueta CODE?

La etiqueta <code> representa un fragmento de código de programación dentro de una línea de texto. Por defecto, los navegadores lo muestran con una fuente monoespaciada.

Descripcion

La etiqueta <code> es un elemento semántico en línea que se utiliza para marcar código de computadora, como nombres de variables, funciones, comandos o fragmentos cortos de código dentro del texto.

Usos apropiados de <code>:

  • Nombres de variables en programación
  • Nombres de funciones o métodos
  • Fragmentos cortos de código inline
  • Comandos de terminal o consola
  • Nombres de archivos o rutas
  • Etiquetas HTML o atributos mencionados en texto

Importante: Para bloques de código multilínea, usa <code> dentro de <pre>. La etiqueta <code> sola es para fragmentos cortos dentro de texto normal.

Atributos principales

La etiqueta <code> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos (útil para resaltado de sintaxis)

El atributo class es comúnmente usado para especificar el lenguaje de programación y aplicar resaltado de sintaxis con librerías como Prism o Highlight.js.

Notas importantes

  • Es un elemento en línea (no genera salto de línea)
  • Se muestra con fuente monoespaciada por defecto
  • Para código multilínea, combinar con <pre>
  • No preserva espacios múltiples ni saltos de línea por sí solo
  • Mejora la semántica al identificar claramente fragmentos de código
  • Es diferente de <kbd> (entrada de teclado) y <samp> (salida de programa)
