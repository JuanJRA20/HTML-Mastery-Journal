La Etiqueta BR

Volver al inicio

¿Qué es la etiqueta BR?

La etiqueta <br> produce un salto de línea (line break) dentro del texto. Es una etiqueta vacía que no requiere etiqueta de cierre.

Descripcion

La etiqueta <br> es un elemento vacío que fuerza un salto de línea en el punto donde se coloca, sin crear un nuevo párrafo. Es útil cuando necesitas dividir líneas dentro de un mismo bloque de contenido.

Usos apropiados de <br>:

  • Direcciones postales (separar líneas de calle, ciudad, código postal)
  • Poemas o versos (respetar la estructura de líneas)
  • Firmas de correo electrónico
  • Separar líneas en formularios de contacto

Uso incorrecto: NO uses múltiples <br> consecutivos para crear espacios verticales. Eso debe hacerse con CSS (margin, padding). Tampoco uses <br> para separar párrafos; para eso está <p>.

Atributos principales

La etiqueta <br> no tiene atributos específicos. Al ser una etiqueta vacía, raramente necesita atributos, aunque técnicamente acepta atributos globales como:

  • id - Identificador único (muy raramente usado)
  • class - Clase CSS (muy raramente usado)

En la práctica, <br> casi nunca lleva atributos.

Ejemplo práctico

<p>
  Calle Mayor 123<br>
  28001 Madrid<br>
  España
</p>

<p>
  Roses are red,<br>
  Violets are blue,<br>
  HTML is great,<br>
  And so are you.
</p>

Notas importantes

  • Es una etiqueta vacía (no tiene </br> de cierre)
  • En XHTML se escribía <br /> pero en HTML5 solo <br> es suficiente
  • NO usar para crear espacios verticales (usa CSS en su lugar)
  • NO usar para separar párrafos (usa <p> en su lugar)
  • Útil solo cuando el salto de línea es parte del contenido, no del diseño
