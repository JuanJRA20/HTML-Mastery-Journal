Las Etiquetas UL, OL y LI

Volver al inicio

¿Qué son las etiquetas UL, OL y LI?

Las etiquetas <ul> (unordered list), <ol> (ordered list) y <li> (list item) se utilizan para crear listas en HTML. <ul> crea listas sin orden específico, <ol> crea listas numeradas, y <li> define cada elemento de la lista.

Descripcion

Las listas son elementos de bloque fundamentales para organizar información relacionada. Existen dos tipos principales:

  • <ul> - Lista desordenada (unordered list): Los elementos se marcan con viñetas (bullets). Se usa cuando el orden de los elementos no importa.
  • <ol> - Lista ordenada (ordered list): Los elementos se numeran automáticamente. Se usa cuando el orden o secuencia es importante.
  • <li> - Elemento de lista (list item): Cada ítem individual dentro de una lista <ul> o <ol>.

Casos de uso comunes:

  • Menús de navegación
  • Características de productos
  • Pasos de instrucciones (ol)
  • Listas de ingredientes o materiales
  • Rankings o clasificaciones (ol)

Atributos principales

type (para <ol>)

Define el tipo de numeración para listas ordenadas. Valores: 1 (números, por defecto), A (letras mayúsculas), a (letras minúsculas), I (romanos mayúsculas), i (romanos minúsculas).

Ejemplo: <ol type="A">

start (para <ol>)

Especifica el número con el que comienza la numeración de la lista. Por defecto es 1.

Ejemplo: <ol start="5"> - La lista comenzará en 5.

reversed (para <ol>)

Invierte el orden de la numeración (cuenta hacia atrás). Es un atributo booleano.

Ejemplo: <ol reversed>

value (para <li> dentro de <ol>)

Establece un valor específico para un elemento de lista ordenada, alterando la secuencia numérica.

Ejemplo: <li value="10"> - Este ítem será número 10.

Ejemplo práctico

<!-- Lista desordenada -->
<ul>
  <li>Manzanas</li>
  <li>Naranjas</li>
  <li>Plátanos</li>
</ul>

<!-- Lista ordenada -->
<ol>
  <li>Abrir el navegador</li>
  <li>Ir a la URL</li>
  <li>Iniciar sesión</li>
</ol>

<!-- Lista con letras -->
<ol type="A">
  <li>Primera opción</li>
  <li>Segunda opción</li>
</ol>

<!-- Listas anidadas -->
<ul>
  <li>Frutas
    <ul>
      <li>Manzanas</li>
      <li>Naranjas</li>
    </ul>
  </li>
  <li>Verduras</li>
</ul>

Notas importantes

  • Solo <li> puede ser hijo directo de <ul> o <ol>
  • Las listas pueden anidarse (listas dentro de listas)
  • No usar listas solo para indentación visual (usa CSS en su lugar)
  • Los navegadores aplican estilos por defecto (viñetas, números, márgenes)
  • Las listas mejoran la accesibilidad al estructurar información relacionada
  • Existe también <dl> (definition list) para listas de términos y definiciones
