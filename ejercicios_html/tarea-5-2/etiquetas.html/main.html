La Etiqueta MAIN

Volver al inicio

¿Qué es la etiqueta MAIN?

La etiqueta <main> representa el contenido principal y único del documento. Contiene la información central de la página, excluyendo contenido repetido como navegación, headers o footers.

Descripcion

La etiqueta <main> es un elemento semántico introducido en HTML5 que identifica el contenido dominante del <body>. Este contenido debe ser único para el documento y no incluir elementos que se repiten en múltiples páginas, como:

  • Barras de navegación
  • Encabezados del sitio
  • Pies de página
  • Barras laterales globales
  • Formularios de búsqueda globales (a menos que sea la función principal de la página)

El contenido dentro de <main> debe ser el foco principal de la página. Por ejemplo, en un blog sería el artículo completo, en una tienda online sería el catálogo de productos, etc.

Atributos principales

La etiqueta <main> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos se utilizan principalmente para aplicar estilos CSS específicos al contenido principal.

Ejemplo práctico

<body>
  <header>
    <h1>Mi Blog</h1>
    <nav>...</nav>
  </header>
  
  <main>
    <article>
      <h2>Título del artículo</h2>
      <p>Contenido principal único de esta página...</p>
    </article>
  </main>
  
  <footer>...</footer>
</body>

Notas importantes

  • Solo puede haber un <main> visible por documento
  • No debe ser descendiente de <article>, <aside>, <footer>, <header> o <nav>
  • Mejora enormemente la accesibilidad, permitiendo a los usuarios saltar directamente al contenido principal
  • Es crucial para SEO, indicando a los motores de búsqueda cuál es el contenido más relevante
  • No es obligatorio, pero es altamente recomendado en HTML5 moderno
