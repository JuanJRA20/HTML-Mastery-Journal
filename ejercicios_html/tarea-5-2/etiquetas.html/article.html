La Etiqueta ARTICLE

¿Qué es la etiqueta ARTICLE?

La etiqueta <article> representa contenido independiente y autocontenido que tiene sentido por sí mismo y podría ser distribuido o reutilizado de forma independiente.

Descripcion

La etiqueta <article> es un elemento semántico introducido en HTML5 que identifica una composición independiente dentro de un documento. El contenido de un <article> debe tener sentido completo si se extrae del contexto de la página.

Ejemplos típicos de uso de <article>:

  • Artículos de blog o noticias
  • Publicaciones en foros
  • Comentarios de usuarios
  • Tarjetas de productos en un catálogo
  • Widgets independientes
  • Entradas de redes sociales

Prueba clave: Si el contenido pudiera sindicarse vía RSS, aparecer en un agregador de noticias, o ser extraído y seguir teniendo sentido completo, probablemente debería estar en un <article>.

Atributos principales

La etiqueta <article> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos son comunes para identificar artículos específicos o aplicar estilos consistentes.

Ejemplo práctico

<article>
  <header>
    <h2>Introducción a HTML5</h2>
    <p>Publicado el 14 de febrero de 2026</p>
  </header>
  <p>HTML5 es la última versión del lenguaje...</p>
  <footer>
    <p>Autor: Juan Pérez</p>
  </footer>
</article>

Notas importantes

  • Puede haber múltiples <article> en una misma página
  • Un <article> puede contener otros <article> (ej: comentarios dentro de un post)
  • Puede contener su propio <header> y <footer>
  • La diferencia con <section>: <article> es independiente, <section> es parte de un todo
  • Mejora el SEO y la accesibilidad al identificar claramente contenido distribuible
