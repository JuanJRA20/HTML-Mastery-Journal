La Etiqueta NAV

Volver al inicio

¿Qué es la etiqueta NAV?

La etiqueta <nav> define una sección de navegación que contiene enlaces a otras páginas o a diferentes partes del mismo documento.

Descripcion

La etiqueta <nav> es un elemento semántico introducido en HTML5 que se utiliza para agrupar los principales bloques de navegación del sitio, como:

  • Menú principal del sitio web
  • Menú de navegación secundaria
  • Tabla de contenidos de un documento largo
  • Breadcrumbs (migas de pan) de navegación
  • Paginación de resultados

Importante: No todos los grupos de enlaces necesitan estar en un <nav>. Esta etiqueta debe reservarse para bloques de navegación principales o importantes. Enlaces secundarios en el footer, por ejemplo, no necesitan estar en un <nav>.

Atributos principales

La etiqueta <nav> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos
  • aria-label - Etiqueta descriptiva para accesibilidad (muy recomendado cuando hay múltiples <nav>)

El atributo aria-label es especialmente útil cuando tienes varios elementos <nav> en la misma página, para diferenciarlos en lectores de pantalla.

Ejemplo práctico

<nav aria-label="Navegación principal">
  <ul>
    <li><a href="inicio.html">Inicio</a></li>
    <li><a href="servicios.html">Servicios</a></li>
    <li><a href="contacto.html">Contacto</a></li>
  </ul>
</nav>

Notas importantes

  • Puede haber múltiples <nav> en un mismo documento
  • No todos los enlaces de la página deben estar en un <nav>
  • Mejora significativamente la accesibilidad para usuarios de lectores de pantalla
  • Ayuda a los motores de búsqueda a identificar la estructura de navegación del sitio
  • Generalmente se coloca dentro de <header> pero puede estar en otras ubicaciones
