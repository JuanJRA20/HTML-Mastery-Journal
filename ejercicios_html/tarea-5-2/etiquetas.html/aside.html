La Etiqueta ASIDE

Volver al inicio

¿Qué es la etiqueta ASIDE?

La etiqueta <aside> representa contenido tangencial o complementario que está indirectamente relacionado con el contenido principal del documento.

Descripcion

La etiqueta <aside> es un elemento semántico introducido en HTML5 que se utiliza para contenido que complementa el contenido principal pero que podría separarse sin afectar la comprensión del mismo.

Usos comunes de <aside>:

  • Barras laterales (sidebars) con información relacionada
  • Llamadas de atención (call-outs) o notas al margen
  • Citas destacadas (pull quotes)
  • Publicidad relacionada con el contenido
  • Biografía del autor en un artículo
  • Enlaces a contenido relacionado
  • Glosarios o definiciones complementarias

Concepto clave: El contenido en <aside> no es esencial para entender el contenido principal, pero aporta valor adicional.

Atributos principales

La etiqueta <aside> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Comúnmente se utilizan para posicionar visualmente el contenido a un lado de la página mediante CSS.

Ejemplo práctico

<main>
  <article>
    <h1>Historia de la Web</h1>
    <p>La World Wide Web fue creada...</p>
  </article>
  
  <aside>
    <h3>Sabías que...</h3>
    <p>Tim Berners-Lee no patentó su invención.</p>
  </aside>
</main>

Notas importantes

  • Puede haber múltiples <aside> en un documento
  • Puede estar dentro de <article> (contenido tangencial al artículo) o fuera (tangencial a toda la página)
  • No usar <aside> para contenido que es parte integral del flujo principal
  • Mejora la accesibilidad identificando claramente contenido secundario
  • Los lectores de pantalla pueden saltar este contenido si el usuario lo desea
