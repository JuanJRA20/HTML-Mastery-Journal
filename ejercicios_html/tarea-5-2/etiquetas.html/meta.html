La Etiqueta META

Volver al inicio

¿Qué es la etiqueta META?

La etiqueta <meta> proporciona metadatos sobre el documento HTML. Estos metadatos no son visibles en la página, pero son leídos por navegadores, motores de búsqueda y redes sociales.

Descripcion

La etiqueta <meta> se coloca siempre dentro del <head> y es una etiqueta vacía (no tiene etiqueta de cierre). Se utiliza para definir información como:

  • Codificación de caracteres del documento
  • Descripción de la página para buscadores
  • Configuración del viewport para diseño responsive
  • Información para redes sociales (Open Graph)
  • Autor y palabras clave del contenido

Atributos principales

charset

Define la codificación de caracteres del documento. Casi siempre se usa UTF-8.

Ejemplo: <meta charset="UTF-8">

name y content

Se usan en combinación para definir metadatos generales. El atributo name especifica el tipo de metadato y content su valor.

Combinaciones comunes:

  • <meta name="description" content="Descripción de la página"> - Para SEO
  • <meta name="keywords" content="html, meta, tutorial"> - Palabras clave (poco usado hoy)
  • <meta name="author" content="Nombre del autor"> - Autor del documento
  • <meta name="viewport" content="width=device-width, initial-scale=1"> - Para diseño responsive

http-equiv y content

Proporciona instrucciones al navegador similares a cabeceras HTTP. Se usa con content para el valor.

Combinaciones comunes:

  • <meta http-equiv="refresh" content="30"> - Refresca la página cada 30 segundos
  • <meta http-equiv="X-UA-Compatible" content="IE=edge"> - Compatibilidad con Internet Explorer

property y content

Se usa para metadatos de Open Graph (redes sociales como Facebook, Twitter). El atributo property define el tipo y content su valor.

Combinaciones comunes:

  • <meta property="og:title" content="Título para redes sociales">
  • <meta property="og:description" content="Descripción para redes sociales">
  • <meta property="og:image" content="url-de-imagen.jpg">

Ejemplo práctico

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Tutorial de HTML5">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <meta property="og:title" content="Mi Sitio Web">
</head>

Notas importantes

  • Es una etiqueta vacía (no tiene </meta>)
  • Puede haber múltiples etiquetas <meta> en un documento
  • La etiqueta <meta charset="UTF-8"> debe ser una de las primeras dentro del <head>
  • El metadato "viewport" es esencial para diseño responsive en dispositivos móviles
  • Los metadatos de descripción influyen directamente en el SEO
