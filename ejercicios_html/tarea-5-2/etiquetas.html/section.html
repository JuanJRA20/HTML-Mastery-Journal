La Etiqueta SECTION

Volver al inicio

¿Qué es la etiqueta SECTION?

La etiqueta <section> define una sección temática del documento. Agrupa contenido relacionado bajo un mismo tema o propósito.

Descripcion

La etiqueta <section> es un elemento semántico introducido en HTML5 que representa una sección genérica de un documento. Cada <section> normalmente tiene su propio encabezado (<h1>-<h6>).

Se utiliza para dividir el contenido en partes temáticas como:

  • Capítulos de un documento largo
  • Pestañas de contenido diferente
  • Secciones numeradas de una tesis
  • Diferentes áreas temáticas de una página (servicios, testimonios, contacto)

Regla general: Si el contenido es independiente y puede existir por sí solo, usa <article>. Si es simplemente una agrupación temática dentro de un contexto mayor, usa <section>.

Atributos principales

La etiqueta <section> no tiene atributos específicos propios. Acepta los atributos globales de HTML5:

  • id - Identificador único del elemento
  • class - Clase CSS para aplicar estilos

Estos atributos son especialmente útiles para aplicar estilos diferenciados a cada sección o para crear enlaces de anclaje.

Ejemplo práctico

<main>
  <section>
    <h2>Nuestros Servicios</h2>
    <p>Ofrecemos diseño web profesional...</p>
  </section>
  
  <section>
    <h2>Testimonios</h2>
    <p>Lo que dicen nuestros clientes...</p>
  </section>
</main>

Notas importantes

  • Puede haber múltiples <section> en un mismo documento
  • Cada <section> debería tener un encabezado (h1-h6)
  • No usar <section> solo como contenedor de estilo (para eso está <div>)
  • Si no puedes darle un título descriptivo a la sección, probablemente no necesites <section>
  • Mejora la estructura semántica y accesibilidad del documento
