¿Qué es la etiqueta HTML?

La etiqueta <html> es el elemento raíz de todo documento HTML. Es el contenedor principal que envuelve absolutamente todo el contenido de una página web.

Descripcion

Todo documento HTML válido debe comenzar con la etiqueta (después de la declaración ) y terminar con . Esta etiqueta indica al navegador que todo lo que está dentro de ella debe interpretarse como código HTML.

La etiqueta contiene únicamente dos elementos hijos directos:

  • <head> - Contiene metadatos y enlaces a recursos externos.
  • <body> - Contiene el contenido visible de la página.

Atributos principales

lang

Especifica el idioma principal del contenido del documento. Es crucial para accesibilidad (lectores de pantalla) y SEO.

dir

Define la dirección del texto: de izquierda a derecha (ltr) o de derecha a izquierda (rtl). Usado para idiomas como árabe o hebreo.

Notas importantes

  • Debe haber solo una etiqueta <html> por documento
  • Siempre debe estar precedida por <!DOCTYPE html> en HTML5
  • Es obligatoria en todos los documentos HTML válidos
  • El atributo lang es técnicamente opcional, pero altamente recomendado
