La Etiqueta IMG

Volver al inicio

¿Qué es la etiqueta IMG?

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es una etiqueta vacía que no requiere etiqueta de cierre.

Descripcion

La etiqueta <img> es un elemento en línea que incrusta una imagen en el documento. A diferencia de los fondos CSS, las imágenes insertadas con <img> son parte del contenido del documento.

Usos comunes de <img>:

  • Fotografías y gráficos del contenido
  • Logotipos de la empresa
  • Ilustraciones y diagramas
  • Iconos que forman parte del contenido (no decorativos)
  • Imágenes de productos en tiendas online
  • Avatares de usuarios

Importante: Las imágenes puramente decorativas deberían manejarse con CSS. La etiqueta <img> es para imágenes que forman parte del contenido y aportan información.

Atributos principales

A continuación se explican en detalle los atributos más importantes:

src

Especifica la ruta o URL de la imagen. Es obligatorio. Puede ser una ruta relativa, absoluta o una URL completa. Sin este atributo, la imagen no se mostrará.

Ejemplo: <img src="imagen.jpg"> o <img src="https://ejemplo.com/foto.png">

alt

Proporciona texto alternativo que describe la imagen. Es esencial para accesibilidad (lectores de pantalla) y SEO. Se muestra si la imagen no carga. Aunque técnicamente es opcional, debe considerarse obligatorio en la práctica.

Ejemplo: <img src="perro.jpg" alt="Perro golden retriever jugando en el parque">

width

Define el ancho de la imagen en píxeles. Especificar dimensiones ayuda al navegador a reservar espacio antes de cargar la imagen, mejorando el rendimiento y evitando saltos de layout.

Ejemplo: <img src="foto.jpg" width="300">

height

Define el alto de la imagen en píxeles. Se recomienda usar junto con width para mantener la proporción correcta y optimizar la carga de la página.

Ejemplo: <img src="foto.jpg" width="300" height="200">

loading

Controla cómo se carga la imagen. Valores: lazy (carga diferida cuando está cerca del viewport, ahorra ancho de banda), eager (carga inmediata, por defecto). Muy útil para optimizar rendimiento.

Ejemplo: <img src="imagen.jpg" loading="lazy">

Otros atributos menos comunes pero útiles:

  • srcset - Especifica múltiples fuentes de imagen para diferentes resoluciones de pantalla (responsive images)
  • sizes - Define tamaños de imagen según el ancho del viewport, trabaja junto con srcset
  • title - Muestra un tooltip al pasar el mouse sobre la imagen

Ejemplo práctico

<!-- Imagen básica -->
<img src="paisaje.jpg" alt="Montañas al atardecer">

<!-- Imagen con dimensiones -->
<img src="logo.png" alt="Logo de la empresa" width="200" height="100">

<!-- Imagen con carga diferida -->
<img src="foto-grande.jpg" 
     alt="Fotografía de la ciudad" 
     width="800" 
     height="600" 
     loading="lazy">

<!-- Imagen responsive -->
<img src="imagen-small.jpg" 
     srcset="imagen-small.jpg 400w, 
             imagen-medium.jpg 800w, 
             imagen-large.jpg 1200w"
     sizes="(max-width: 600px) 400px, 
            (max-width: 900px) 800px, 
            1200px"
     alt="Imagen adaptable">

Notas importantes

  • Es una etiqueta vacía (no tiene </img> de cierre)
  • El atributo alt es crucial para accesibilidad y SEO
  • Especificar width y height previene saltos de contenido al cargar
  • Usa loading="lazy" para imágenes fuera del viewport inicial
  • Formatos comunes: JPG (fotos), PNG (transparencias), SVG (vectores), WebP (moderno, mejor compresión)
  • Las imágenes deben optimizarse antes de subirse (comprimir, dimensiones adecuadas)
