La Etiqueta BODY

Volver al inicio

¿Qué es la etiqueta BODY?

La etiqueta <body> contiene todo el contenido visible de una página web. Todo lo que el usuario ve en su navegador (textos, imágenes, videos, formularios, etc.) debe estar dentro de esta etiqueta.

Descripcion

La etiqueta <body> se coloca después del <head> y antes del cierre de </html>. Es el segundo hijo directo de la etiqueta <html>.

Dentro del <body> se colocan todos los elementos visibles como:

  • Encabezados (<h1> a <h6>)
  • Párrafos (<p>)
  • Imágenes (<img>)
  • Enlaces (<a>)
  • Formularios (<form>)
  • Elementos semánticos (<header>, <nav>, <footer>, etc.)

Atributos principales

En HTML5 moderno, la etiqueta <body> no tiene atributos específicos de uso común. Los atributos de eventos que existían en versiones anteriores (como onload, onunload) ahora se manejan mejor con JavaScript.

Acepta atributos globales como id y class, que sí se usan ocasionalmente para aplicar estilos CSS al cuerpo completo del documento.

Ejemplo práctico

<body>
  <h1>Bienvenido a mi sitio</h1>
  <p>Este es el contenido visible.</p>
  <img src="foto.jpg" alt="Descripción">
</body>

Notas importantes

  • Debe haber solo una etiqueta <body> por documento
  • Es obligatoria en todo documento HTML válido
  • Todo el contenido visible debe estar dentro del <body>
  • Los scripts pueden colocarse al final del <body> para mejorar el rendimiento de carga
