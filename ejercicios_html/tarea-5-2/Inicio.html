HTML

HTML, son las siglas de HyperText Markup Language, que puede traducirse como lenguaje de marcas o marcas de hiper texto

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro, y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener el mismo nombre en general.

HTML se compone de etiquetas que se escriben entre simbolos menor que y mayor que

Historia de HTML

Los inicios del lenguaje HTML se remontan a inicios de 1990, cuando Tim Berners-Lee creo la primera pagina web.

Versiones de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991.

En la actualidad, la ultima version de HTML es HTML5

Si quiere saber mas acerca de las principales versiones de HTML, aqui puede seleccionar los que quiera ver:

HTML 2.0 (1995)

HTML 3.2 (1997)

HTML5 (2014)

Lista de etiquetas

Para esta lista de etiquetas, tendremos en cuenta la ultima version de HTML, que es HTML5. Puedes pinchar en cada etiqueta para obtener mas informacion de la misma:

Estructura fundamental:

  • <html> - Raíz del documento, contiene todo el contenido HTML
  • <head> - Metadatos del documento (título, enlaces a CSS, scripts)
  • <body> - Contenido visible de la página
  • <title> - Título que aparece en la pestaña del navegador
  • <meta> - Metadatos (charset, viewport, descripción SEO)

Estructura semántica HTML5:

  • <header> - Encabezado de página o sección
  • <nav> - Sección de navegación con enlaces
  • <main> - Contenido principal de la página
  • <section> - Sección genérica de contenido
  • <article> - Contenido independiente y autocontenido
  • <aside> - Contenido secundario o complementario
  • <footer> - Pie de página
  • <address> - Información de contacto del autor/propietario

Títulos y párrafos:

  • <h1> a <h6> - Títulos jerárquicos (h1 más importante)
  • <p> - Párrafo de texto

Enlaces y contenedores:

  • <a> - Hipervínculo a otras páginas o recursos
  • <ul> / <ol> / <li> - Listas desordenadas/ordenadas y sus elementos
  • <div> - Contenedor genérico sin semántica

Formato de texto básico:

  • <strong> - Importancia fuerte (normalmente negrita)
  • <em> - Énfasis semántico (normalmente cursiva)
  • <br> - Salto de línea forzado
  • <wbr> - Sugerencia de punto de ruptura de palabra (word break opportunity)
  • <span> - Contenedor en línea para estilizar texto

Citas y código:

  • <cite> - Referencia al título de una obra (libro, película, artículo)
  • <code> - Fragmento de código en una línea
  • <pre> - Texto preformateado que respeta espacios y saltos
  • <time> - Marca semántica de fechas/horas legibles por máquinas
  • <blockquote> - Cita en bloque

Multimedia HTML5:

  • <img> - Imagen
  • <audio> - Reproductor de audio
  • <video> - Reproductor de video
  • <canvas> - Área para gráficos dinámicos

Formularios e interacción:

  • <form> - Formulario para enviar datos
  • <input> - Campo de entrada (texto, checkbox, radio, etc.)
  • <textarea> - Área de texto multilinea
  • <button> - Botón interactivo
  • <select> - Menú desplegable
  • <label> - Etiqueta para elementos de formulario

Otros elementos clave:

  • <script> - Inserta o referencia código JavaScript
