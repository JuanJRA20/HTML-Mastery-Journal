HTML 4.01 (1999)

Volver al inicio

HTML 4.01, estandarizado por el W3C en 1999, es una revisión sólida del lenguaje HTML 4.0 enfocada en la internacionalización, accesibilidad y separación de la estructura del contenido del diseño mediante hojas de estilo (CSS). Permite la inclusión de scripts y objetos, facilitando páginas dinámicas y accesibles.

Aspectos clave de HTML 4.01

Tipos de Documento (DTD):

Define tres tipos: Strict (sin elementos de presentación), Transitional (más flexible) y Frameset (para marcos).

Mejoras:

Mejoró el soporte de tablas, formularios, scripts y la internacionalización, permitiendo sitios en cualquier idioma.

Accesibilidad:

Diseñado para cumplir con estándares para usuarios con discapacidades.

Separación:

Fomenta el uso de CSS para el diseño, liberando al HTML de tareas de presentación.

Sintaxis:

Es una aplicación SGML, lo que requiere una declaración DOCTYPE específica para el tipo de DTD.

Aunque fue el estándar más utilizado durante la década de 2000, ha sido mayormente reemplazado por HTML5, que introdujo elementos semánticos más avanzados y soporte nativo multimedia.
