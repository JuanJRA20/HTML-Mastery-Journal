HTML 2.0 (1995)

Volver al inicio

HTML 2.0, aprobado en noviembre de 1995 como RFC 1866 por la IETF, fue el primer estándar oficial del lenguaje de marcado para la World Wide Web. Diseñado para la compatibilidad entre navegadores, introdujo características esenciales como formularios (<form>), imágenes incrustadas (<img>), y soporte para subida de archivos.

Características y detalles clave:

Estandarización:

Basado en la práctica común de junio de 1994, formalizó la estructura básica de la web.

Funcionalidades:

Incluía elementos estructurales (<head>, <body>), enlaces (<a>), listas, y el inicio de los formularios (campos de texto, botones).

Enfoque:

Enfocado en la estructura semántica y la funcionalidad hipertextual, no en el diseño visual.

Limitaciones:

Carecía de soporte nativo para tablas avanzadas y estilos (CSS), lo que impulsó la creación de etiquetas propietarias por navegadores como Netscape.

Desarrollo:

Desarrollado por el grupo de trabajo HTML de la IETF, con Dan Connolly como uno de los editores principales.

HTML 2.0 estableció la base técnica para la web, promoviendo la interoperabilidad antes de la llegada de HTML 3.2 y las hojas de estilo.
